---

## :computer: Prérequis

*Nécessite l'installation de **[Go ~1.19](https://go.dev/dl/)**

``` 
git clone https://github.com/ClrGe/negosud-gui.git
```
```
cd negosud-gui && go mod tidy
```
Compiler le programme :
```
go run main.go
```
Créer un exécutable :
```
go build
```
